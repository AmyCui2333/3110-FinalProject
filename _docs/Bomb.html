Module Bomb

module Bomb: sig .. end

Representation of bombs planted by the player.

type t

The abstract type of values representing the bomb with position, power, and start time.

val make_bomb : Player.t -> t option

make_bomb p makes the bomb at the player's current position with the bomb_power initiated from player and record a start time of bomb.

val get_neighbours : int -> t -> (int * int) list -> (int * int) list

get_neighbours pwr b lst returns the position of all neighboring tiles affectted by the bomb, depending on the bomb's power pwr.

val check_explode : t -> bool

check_explode b returns true if it's time for the bomb b to de exploded else false.

val explode : Background.t -> t list -> Background.t

explode bkg b_lst explode all bombs in b_lst on the background. background stays the same if there's no bomb explosion.

val get_pos : t -> int * int

get_pos t returns the postion of the bomb.

val in_blast_area : t -> int * int -> bool

in_blast_area b pos returns true if position pos is within bomb n's blast area.

val in_blast_lst : t list -> int * int -> bool

in_blast_lst bomb_list pos returns true if position pos is within the blast area of any of the bombs in bomb_list.

val show_tools : t list -> Background.t -> int -> (int * int) list

show_tools b_lst bkg i returns a list of tool_i's positions in the background map bkg; all tool_i in the returned list were hidden in bushes before exploded by bombs listed in b_lst.
