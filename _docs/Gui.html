Module Gui

module Gui: sig .. end

Using library module "Graphic_image" to display Game GUI.

val draw_canvas : unit -> unit

draw_canvas () initializes the window for this game.

val draw_heart_on_board : Player.t -> unit

draw_heart_on_board plr draws hearts on the left panel based on how many lives plr has left.

val draw_state : State.t -> unit

draw_state st draws the player, enemy, and obstacles of a state st.

val draw_move : State.t -> int * int -> (int * int) option -> unit

draw_move st plr_pos enemy_pos covers up the grids at plr_pos and enemy_pos, which are the player and enemy's previous position. It then draws the new player and enemy according to st.

val draw_bomb : int * int -> unit

draw_bomb bomb_pos draws a bomb at the position bomb_pos.

val draw_explosions : Bomb.t list -> State.t -> Player.t -> unit

draw_explosions draws the animation of bomb explosion.

val clear_tools : State.t -> unit

clear_tools erases the tools taken in by the player.

val draw_tools : State.t -> unit

draw_tools draws all tools bombed out from bombs if any.

val draw_score : State.t -> unit

draw_score covers up the previous score on the left panel and draws the current score stored in state st.

val draw_ending : State.t -> bool -> unit

draw_ending draws the ending graph of game, displays the player's final score and a piece of information representing whether the player wins. Exits the game 7 seconds after this screen.

val draw_choose : unit -> unit

draw_choose draws the choose-player screen and allows users to pick charater. Displays two profile pictures of "camel" and "lama" from which the player can choose as their avator.

val draw_instruction : unit -> unit

draw_instruction draws the opening page as the start of the game, displaying instructions for the player.

val draw_left_panel : State.t -> unit

draw_left_panel draws a panel at the left side of game gui, displaying the player's score, health (hearts), and a profile picture of either "camel" or "lama" according to the player's choice. Initializes the left panel with initial health (3 hearts) and intila score (0).
