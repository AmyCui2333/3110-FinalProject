Module ToolSpeedUp

module ToolSpeedUp: sig .. end

Representation of tool of type 1 "Speed-up".

type t

The abstract type of values representing the ToolSpeedUp.

val get_speedup_xy : t -> int * int

get_speedup_xy tool returns the position of ToolSpeedUp tool.

val new_speedups_fromxy : (int * int) list -> t list

new_speedups_fromxy pos_lst returns a list of ToolSpeedUp from the position list pos_lst.

val xy_to_speedup : int * int -> t list -> t

xy_to_speedup pos tool_lst returns the ToolSpeedUp that is located at position pos from the list of ToolSpeedUp tool_lst.

val speedup_plr : t -> Player.t -> Player.t

speedup_plr tool plr returns a player plr whose speed is changed by ToolSpeedUp tool.
